# Act(막) 문서 평가 기준

## 1. 평가 목적과 활용 방법

### 평가 목적

Act 문서는 작품의 골격을 이루는 최상위 구조 단위 문서이다. 이 평가 기준은 다음 세 가지 목적을 위해 존재한다.

1. **품질 게이트**: Act 문서가 하위 문서(Chapter, Scene) 작성의 근거로 기능할 수 있는 수준인지 판단한다. Act 문서의 구조적 결함은 하위 문서에서 증폭되므로, 초기 단계에서의 품질 확보가 핵심이다.

2. **일관성 검증**: 6개 Act 문서(프롤로그, 1-4막, 에필로그)가 하나의 작품으로서 통일된 방향성을 유지하는지 확인한다. 개별 문서가 아무리 훌륭해도 전체 흐름에서 이질적이면 의미가 없다.

3. **협업 품질 관리**: AI-인간 협업 환경에서, 서로 다른 시점에 서로 다른 주체가 작성한 문서들이 동일한 품질 기준을 충족하도록 보장한다.

### 활용 방법

| 활용 시점 | 방법 |
|-----------|------|
| Act 문서 신규 작성 직후 | 전체 항목을 체크리스트로 활용하여 누락 여부 점검 |
| Chapter 문서 작성 전 | 해당 Act 문서가 B등급 이상인지 확인 후 진행 |
| 전체 작품 구조 리뷰 시 | 6개 Act 문서를 병렬로 놓고 횡단 평가 실시 (부록 참조) |
| Act 문서 수정/갱신 후 | 변경된 항목 중심으로 재평가 |

## 2. 평가 항목별 기준

### 항목 A: 구조적 완결성 (배점 25점)

Act 문서가 필수 섹션을 빠짐없이 포함하고, 각 섹션이 제 기능을 다하는지 평가한다.

#### 필수 섹션 체크리스트

| 섹션 | 기능 | 확인 |
|------|------|------|
| 막의 주요 목표 | 이 막이 전체 작품에서 수행하는 서사적 역할 명시 | [ ] |
| 포함된 장 목록 | 하위 Chapter 문서 파일명과 제목 매핑 | [ ] |
| 주요 전환점 | 막 내에서 일어나는 핵심 사건의 인과 관계 | [ ] |
| 감정적 아크 | 시작-중간-전환-끝의 감정 흐름도 | [ ] |
| 막 요약 | 주요 사건 흐름, 주요 캐릭터, 막의 의미 | [ ] |
| 다음 막으로의 연결 | 후속 막으로의 브릿지 정보 | [ ] |
| 분위기와 톤 | 시작/중반/후반/전체 톤 가이드 | [ ] |
| 주요 모티프 | 이 막에서 활용되는 모티프와 상징 | [ ] |
| 공간 동선 요약 | 이 막의 주요 무대와 공간 이동 경로 | [ ] |
| 시간 프레임 | 이 막의 사건이 벌어지는 대략적 시간 범위 | [ ] |
| 참조 문서 | Chapter, Scene 문서 목록 (상대적 비중 표시 포함) | [ ] |

**프롤로그/에필로그 형식 예외**: 프롤로그와 에필로그는 다른 막과 구조가 상이할 수 있다. 프롤로그는 "포함된 장 목록" 대신 씬 단위로 기술할 수 있으며, "다음 막으로의 연결"을 "복선" 섹션으로 대체할 수 있다. 에필로그는 "다음 막으로의 연결"이 해당 없음이며, Scene 문서 없이 직접 집필 가능하다. 단, 나머지 섹션은 동일하게 포함해야 한다.

#### 세부 기준

| 세부 기준 | 최대 배점 |
|-----------|-----------|
| 필수 섹션 완비 여부 | 5점 |
| A-1 목표의 명확성 | 6점 |
| A-2 전환점의 인과성 | 6점 |
| A-3 감정 아크의 구체성 | 4점 |
| A-4 참조 문서의 정합성 | 4점 |
| 합계 | 25점 |

- **[A-1] 목표의 명확성**: 막의 주요 목표가 3-5개의 구체적 항목으로 서술되어 있으며, 각 목표가 측정 가능한 서사적 성과와 연결되는가.
  - 좋은 예: "악당과의 첫 대면으로 긴장감 조성" (구체적, 검증 가능)
  - 나쁜 예: "이야기를 진행시킨다" (모호, 검증 불가)

- **[A-2] 전환점의 인과성**: 주요 전환점뿐 아니라 막 내 주요 사건 전반이 "원인 -> 사건 -> 결과"의 인과 관계로 서술되어 있는가. 전환점 사이의 연결 사건(connective tissue)도 설득력 있는 인과를 갖추어야 한다.

- **[A-3] 감정 아크의 구체성**: "시작-중간-전환-끝" 4단계가 모두 있으며, 각 단계에서 독자가 느낄 구체적 감정이 명시되어 있는가.

- **[A-4] 참조 문서의 정합성**: 나열된 Chapter/Scene 문서 파일명이 프로젝트의 넘버링 규칙과 일치하며, 각 Chapter/Scene의 상대적 비중 또는 예상 분량이 표시되어 있는가.

### 항목 B: 서사적 기능 (배점 25점)

이 막이 전체 4막 구조(기승전결) 안에서 자기 역할을 올바르게 수행하는지 평가한다.

#### 세부 기준

| 세부 기준 | 최대 배점 |
|-----------|-----------|
| B-1~B-6 막별 고유 기능 (해당 막 기준 적용) | 20점 |
| 분량 비중 적합성 | 5점 |
| 합계 | 25점 |

#### 막별 고유 기능 기준

**[B-1] 프롤로그**
- [ ] 악당의 위협을 효과적으로 제시하되, 본편 톤(모험-코미디)과의 연결 고리가 있는가
- [ ] 본편에서 활용될 복선이 2개 이상 심어져 있는가
- [ ] 프롤로그만의 독립적인 서사 완결성이 있는가 (그 자체로 짧은 이야기가 되는가)
- [ ] 전체 분량의 5% 수준(2,000-2,500자)에 맞는 밀도인가

**[B-2] 제1막 (기/설정)**
- [ ] 5명의 주인공 전원이 소개되며, 각자의 핵심 특성이 1개 이상의 행동/대사로 드러나는가
- [ ] 중심 갈등(마을 수몰 위기)이 명확히 제시되는가
- [ ] 모험의 동기가 자연스럽게 부여되는가 (전설 -> 희망 -> 결심의 흐름)
- [ ] "평범한 일상 -> 위기 인식 -> 행동 결심"의 3단 전환이 있는가
- [ ] 전체 분량의 20% 수준에 맞는 밀도인가

**[B-3] 제2막 (승/전개)**
- [ ] 제1막에서 설정한 갈등이 구체적 위험으로 실체화되는가 (악당 대면, 물리적 위기)
- [ ] 판타지 요소(도깨비 두루)가 자연스럽게 도입되며, 세계관의 규칙이 명시되는가
- [ ] 미드포인트(중간 전환점)가 명확히 존재하며, 이야기의 방향을 바꾸는가
- [ ] 캐릭터들의 위기 대응 방식이 각자의 설정된 성격과 일치하는가
- [ ] 전체 분량의 30% 수준에 맞는 밀도인가 (가장 큰 비중)

**[B-4] 제3막 (전/위기와 성장)**
- [ ] 팀 완성(5인 합류)이 이루어지는가
- [ ] 팀 내부 갈등의 축이 명확히 설정되는가 (어떤 캐릭터 간에, 어떤 이유로)
- [ ] 각 캐릭터의 고유 재능이 최소 1회 이상 발휘되는 장면이 설계되어 있는가
- [ ] 갈등 -> 극복 -> 신뢰 구축의 흐름이 있는가
- [ ] 전체 분량의 25% 수준에 맞는 밀도인가

**[B-5] 제4막 (결/클라이맥스)**
- [ ] 핵심 수수께끼('욕심 없는 자만 열 수 있는 상자')의 해답이 서사적으로 풀리는가
- [ ] 악당의 패배가 자업자득 구조인가 (탐욕이 스스로를 파멸시키는 구조)
- [ ] 도깨비 두루의 희생/도움이 감정적 클라이맥스를 형성하는가
- [ ] 진정한 보물의 정체가 주제적으로 설득력 있게 드러나는가
- [ ] 감정의 진폭(긴장 최고조 -> 따뜻한 해결)이 작품 전체에서 가장 큰가
- [ ] 전체 분량의 15% 수준에 맞는 밀도인가 (압축적 긴장)

**[B-6] 에필로그**
- [ ] 모험의 물리적 결과(마을 구원)와 내면적 성장이 함께 정리되는가
- [ ] 5명 각 캐릭터의 성장이 모험 전후 대비로 압축적으로 제시되는가
- [ ] 악당의 결말이 간결하게 처리되는가
- [ ] 열린 결말의 여운이 있는가 (두루와의 관계, 마을의 미래)
- [ ] 전체 분량의 5% 수준에 맞는 밀도인가

### 항목 C: 캐릭터 운용 (배점 20점)

#### 세부 기준

| 세부 기준 | 최대 배점 |
|-----------|-----------|
| C-1 등장인물 명시 | 3점 |
| C-2 역할 분배의 균형 | 4점 |
| C-3 성격 일관성 | 4점 |
| C-4 성장 아크 진행 | 3점 |
| C-5 악당 캐릭터의 입체성 | 3점 |
| C-6 캐릭터 상호작용 설계 | 3점 |
| 합계 | 20점 |

- **[C-1] 등장인물 명시**: 이 막에서 활약하는 주요 캐릭터가 이름과 역할로 명시되어 있는가.

- **[C-2] 역할 분배의 균형**: 특정 캐릭터에 비중이 과도하게 쏠리지 않으며, 각자가 서사적으로 기여하는 바가 있는가.
  - 평가 방법: 각 캐릭터의 "이 막에서의 기능"을 한 문장으로 요약할 수 있어야 한다. 요약할 수 없는 캐릭터는 비중이 부족한 것이다.

- **[C-3] 성격 일관성**: 캐릭터의 행동/반응이 character-overview.md에 정의된 핵심 특성, 성격 키워드와 일치하는가.
  - 체크 포인트:
    - 임수진: 공적/사적 모드 전환이 나타나는가
    - 박지훈: 자연/동물과의 교감, 사회적 필터 없는 반응이 있는가
    - 고민준: 콘텐츠/촬영에 대한 집착이 드러나는가
    - 박지영: 열정적이고 직진형인 행동 패턴이 있는가
    - 강태호: 논리적 분석과 감정 억제가 나타나는가

- **[C-4] 성장 아크 진행**: 이 막이 각 캐릭터의 전체 성장 아크에서 어느 지점에 해당하며, 이전 막 대비 어떤 변화가 있는지 암시되어 있는가.

- **[C-5] 악당 캐릭터의 입체성**: 악당이 단순한 장애물이 아니라, 동기와 관계 역학(형제 관계)이 드러나는가.

- **[C-6] 캐릭터 상호작용 설계**: 이 막에서 부각되는 주요 캐릭터 조합(케미)과 그 상호작용의 서사적 기능이 명시되어 있는가. (예: "수진-지훈-민준 삼인조의 위기 대응", "민준-두루의 초콜릿 우정")

### 항목 D: 톤과 분위기 설계 (배점 15점)

#### 세부 기준

| 세부 기준 | 최대 배점 |
|-----------|-----------|
| D-1 톤 가이드의 구체성 | 3점 |
| D-2 모험/긴장 대 우정/유머 비율 | 3점 |
| D-3 연령 적합성 | 2점 |
| D-4 유머 요소 설계 | 2점 |
| D-5 페이싱과 감정 리듬 | 3점 |
| D-6 독자 경험 설계 | 2점 |
| 합계 | 15점 |

- **[D-1] 톤 가이드의 구체성**: "시작/중반/후반/전체"로 구분된 톤 가이드가 있으며, 각 구간에서 의도하는 분위기가 구체적 단어로 서술되어 있는가.
  - 좋은 예: "동굴에 갇힌 상태의 공포(긴장감 수준, 실제 공포 아님)"
  - 나쁜 예: "무서운 분위기"

- **[D-2] 모험/긴장 대 우정/유머 비율**: 이 막의 톤 비율이 명시되어 있으며, 전체 작품의 기본 비율(60:40)과의 차이가 설명되어 있는가. 비율이 기본과 다르다면 그 이유가 서사적으로 정당한가.

- **[D-3] 연령 적합성**: 공포/폭력 요소가 "긴장감" 수준을 넘지 않도록 명시적으로 가이드하고 있는가. 어린이 독자 고려사항이 별도로 언급되어 있는가.

- **[D-4] 유머 요소 설계**: 캐릭터별 반복 코미디 패턴(고민준의 촬영 집착, 수진의 허당 노출 등)의 활용이 계획되어 있는가.

- **[D-5] 페이싱과 감정 리듬**: 긴장과 이완이 교차하는 리듬이 설계되어 있으며, 각 전환의 서사적 트리거(사건, 캐릭터 행동, 발견 등)가 명시되어 있고, 독자가 숨을 돌릴 수 있는 구간이 확보되어 있는가. 이 막의 전반적인 페이싱 성격(예: "전반은 느린 탐색, 후반은 빠른 추격")이 제시되어 있는가.

- **[D-6] 독자 경험 설계**: 이 막의 도입 후크(독자의 관심을 붙잡는 장치), 막 끝의 페이지 터닝 요소(다음 막을 읽고 싶게 만드는 장치), 그리고 독자가 가장 감정적으로 투자하게 되는 앵커 장면이 설계되어 있는가.

### 항목 E: 연결성과 일관성 (배점 15점)

#### 세부 기준

| 세부 기준 | 최대 배점 |
|-----------|-----------|
| E-1 이전 막과의 연결 | 2점 |
| E-2 다음 막으로의 브릿지 | 3점 |
| E-3 모티프 연속성 | 3점 |
| E-4 설정 정합성 | 3점 |
| E-5 분량 비중 적절성 | 2점 |
| E-6 복선 설계 | 2점 |
| 합계 | 15점 |

- **[E-1] 이전 막과의 연결**: 이 막의 시작점이 이전 막의 끝과 자연스럽게 이어지는가. 시간적/공간적/감정적 연속성이 있는가.

- **[E-2] 다음 막으로의 브릿지**: "다음 막으로의 연결" 섹션이 있으며, 후속 막의 핵심 사건을 자연스럽게 유도하는 떡밥/미해결 과제가 제시되어 있는가.

- **[E-3] 모티프 연속성**: 이 막에서 사용하는 모티프가 작품 전체의 주요 모티프(청동 거울, 꽃이 피는 돌, 욕심 없는 자만 열 수 있는 상자, 구름골)와 연결되며, 이전 막에서의 모티프 활용과 일관적인가.

- **[E-4] 설정 정합성**: 용어, 지명, 인물명, 시간대, 장소 등이 다른 Act 문서 및 상위 문서와 일치하는가.

  - **설정 충돌 해결 규칙**: Act 문서에 상위 문서(plot-structure.md, README.md 등)에 없는 새로운 설정/규칙이 등장한 경우, (a) 상위 문서에 반영하여 공식화하거나 (b) Act 문서에서 삭제해야 한다. Act 문서가 독자적으로 새 규칙을 만들어 방치하는 것은 불허한다.

## 부록: 횡단 평가 기준

### 횡단 평가의 위상

횡단 평가는 개별 Act 문서 평가가 모두 완료된 후, 6개 Act 문서를 병렬로 놓고 실시하는 **별도의 검증 단계**이다.

- **실시 시점**: 개별 Act 문서가 모두 B등급 이상을 달성한 후 실시
- **개별 등급과의 관계**: 횡단 평가 결과는 개별 Act의 등급을 직접 변경하지 않는다. 대신, 횡단 평가에서 발견된 문제는 해당 Act 문서의 수정 요청으로 이어지며, 수정 후 개별 재평가를 통해 반영된다.
- **합격 기준**: 횡단-1~5 각 항목에서 "심각한 불균형" 또는 "구조적 공백"이 발견되지 않아야 한다. 발견 시, 관련 Act 문서 수정 후 횡단 재평가를 실시한다.

### 횡단-1: 긴장 곡선의 적절성

6개 Act 문서의 감정적 아크를 이어 그렸을 때, 전체적인 긴장 곡선이 자연스러운 상승-하강-상승-최고조-해소의 흐름을 그리는가.

아래 곡선은 **독자의 감정적 몰입도**(서사적 위험도가 아니라, 독자가 느끼는 긴장과 감정적 투자의 정도)를 측정한다. 프롤로그는 긴장감 있는 장면이지만 독자의 캐릭터에 대한 감정적 투자가 아직 낮으므로 중간 수준에서 시작한다.

```
감정적 몰입도
  ^
  |           **** (4막: 클라이맥스)
  |         **    **
  |   *   **        ** (에필로그: 해소)
  |  * * *  (3막: 갈등)
  | *   *
  |* (2막: 모험)
  |* (1막: 설정)
  *--(프롤로그)--+--1막--+--2막--+--3막--+--4막--+--에필---> 진행
```
