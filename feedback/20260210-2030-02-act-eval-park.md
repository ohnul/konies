### 가장 까다로운 부분: 추격전 (02-04-03)

**기술적으로 가장 복잡한 이유:**

1. **동시 동선 관리**: 5명의 도주자(수진/지훈/민준/겨울이/두루) + 2명의 추격자(마기재/마희재) = 총 7개 캐릭터 관리
2. **두루의 이동 규칙 통합**: "빛이 있으면 활동, 없으면 불가" 규칙을 자연스럽게 액션에 통합
3. **코미디-액션 교차**: 마희재 끼임 → 겨울이 유인 → 폭포 긴장 → 탈출 카타르시스의 4가지 톤 전환
4. **물리적 공간의 설득력**: 동굴 내부를 독자가 머릿속으로 그릴 수 있어야 함

**해결 방안:**
- Chapter: 동굴 내부 지도/텍스트 배치 확정
- Scene: 추격전 비트를 초 단위까지 설계
- Scene: 두루 빛 규칙의 전술적 활용 메커니즘 확정

### 앵커 장면(두루 소환-교감) 집필의 4대 위험 요소

**위험 요소 1: 공포→교감 감정 전환 속도**
- 문제: 너무 빠르면 수용 불가, 너무 느리면 지루함
- 권장 비율: 공포 60% : 전환 15% : 교감 25%
- 권장 구조: 4개 비트로 분해
  - 비트 1 (공포): 그림자 솟아오름, 노란 눈 (3-4문단)
  - 비트 2 (대치): 양쪽 정지, 침묵, 두루의 떨림 (1-2문단)
  - 비트 3 (전환): 지훈의 "이 친구도 우리만큼 무서운 거야" (1문단)
  - 비트 4 (교감): 민준의 초콜릿, "달다!" (2-3문단)

**위험 요소 2: 두루의 "첫인상" 묘사**
- 현황: "거대한 몸집, 시커먼 피피, 노란 눈" 3요소만
- 부족: 키/체형/피부 질감/얼굴(뿔/이빨?)/의복/특수 표식/동작 특성
- 위험: 캐릭터 상세 문서가 없으면 문서마다 다른 두루가 그려짐
- 해결: Chapter 단계에서 외형 상세 확정 필수

**위험 요소 3: 청동 거울 소환 메커니즘의 시각적 연출**
- 현황: "거울 표면에서 빛이 번지면서 거대한 그림자가 솟아오르며 노란 눈 두 개" (1줄)
- 부족: 시각/청각/촉각의 구체적 단계별 연출
- 권장 시퀀스 (3-5단계):
  1. 빛이 거울에 닿음 → 거울 표면이 물결치듯 일렁임 (시각)
  2. 낮은 웅웅거림이 동굴에 울림 (청각)
  3. 거울 주변 공기가 따뜻해짐 (촉각)
  4. 거울 속에서 그림자가 솟아오르기 시작 (시각)
  5. 완전한 형체 -- 두루의 등장 (크기 충격)

**위험 요소 4: NOTE-TO-AI 미해결**
- 번호판 인식: 2막 사건 촉발의 핵심, 해결 필수
- 거울 이동 규칙: 2막 추격전에 영향, 범위 확정 필요

| 마기재 | 보물 탐욕, 지적 허영 | B+ | 보물 탐색의 구체적 목표 약간 모호 |

**가장 심각한 공백: 추격전 중의 조명**

### 핵심 오브젝트 묘사

**청동 거울:**
- 크기: 미확정 (손바닥? 얼굴 크기? 방패 크기?)
- 형태: 미확정 (원형? 사각? 손잡이?)
- 무게: 미확정 (아이가 들고 뛸 수 있는가?)
- 집필 영향: 크기와 무게가 확정되지 않으면 추격전 설득력 약함

**인광 광물:**
- 색깔: 미확정 (녹색? 청색? 백색?)
- 밝기: "희미한" 정도만 (방향 감각 유지 수준? 겨우 인식 수준?)
- 분포: 동굴 전체? 제단 주변만?

**두루의 외형:**
- "거대한 몸집, 시커먼 피부, 노란 눈" 3요소 외에는 미확정

## 6. 에너지 투입 우선순위

4. **02-04-01 (겨울이 구출)** -- **중**: 감동적이나 구조 단순

6. **02-04-03 끝 (막 종결)** -- **중**: 후크 삽입의 자연스러움

### 긴장-이완 비트별 톤 전환 난이도

| 전환 | 트리거 | 난이도 |
|------|--------|--------|
| 기대 → 불안 | 수상한 차량 발견 | B (중) |
| 불안 → 공포 | 민준 카메라 떨어뜨림 | A (상) |
| 공포 → 안도 | 겨울이 등장 | B+ (중상) |
| 안도 → 호기심 | 인광 빛 발견 | B (중) |
| 호기심 → 경이 | 두루 소환 | **S (최상)** |
| 경이 → 교감 | 초콜릿 | A (상) |
| 교감 → 추격 스릴 | 악당 발소리 | B+ (중상) |
| 스릴 → 코믹 이완 | 마희재 끼임 | A (상) |
| 코믹 → 아찔함 | 폭포 장애물 | B (중) |
| 아찔함 → 해방감 | 동굴 탈출 | B (중) |

**전체**: S급 1개, A급 3개, B+급 2개, B급 4개 = **상위 40%가 고난도**
