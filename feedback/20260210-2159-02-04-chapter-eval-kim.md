**[B-2] 정보 공개 전략 (5/5)**

"정보 공개 전략" 섹션이 독립적으로 존재하며, 6개 정보 항목에 대해 씬/전달 방식/전달자를 매핑한 테이블이 제공된다.

"info dump 방지 방안" 하위 섹션에서 3개 핵심 정보(두루의 설정, 동굴 구조, 세계관 규칙)의 점진적 공개 전략이 구체적으로 기술:
- 두루의 설정: 소환(외형) -> 공포(크기/피부) -> 교감(성격/말투) -> 추격전(규칙/능력) 순차 공개

3. 지훈의 소통과 연결: "소통은 말이 아니라 마음"이라는 발견의 시작
4. 겨울이의 역할 격상: "구출자 + 전술적 유인자"로 결정적 활약
5. 두루의 성장 아크 시작: 인간 불신이 완화되는 첫 경험

각 서브플롯에 관련 씬 번호가 매핑되어 있으며, 동사+결과 형태가 정확하다. 02-03-chapter.md(씬 매핑 있었으나 3개 서브플롯)에서 02-04(5개 서브플롯 + 씬 매핑)로 서브플롯 관리가 한층 정교해졌다. [+5]

**[B-4] 미니 클라이맥스와 서사적 완결감 (4/5)**

미니 클라이맥스가 씬 2의 "두루 소환 -> 공포 -> 교감 전환 순간"으로 명시되어 있으며, 그 효과가 상세히 서술:
- 3장 미니 클라이맥스(캐릭터 내면 전환)와의 질적 차이
- 작품 전체 클라이맥스(선택과 희생)와의 스케일 차이
- "놀라움과 따뜻함" vs "결단과 슬픔"이라는 감정의 질적 차이까지 분석

장 전체의 서사 흐름(구출 -> 탐색 -> 발견 -> 교감 -> 추격 -> 탈출)이 자족적 구조를 가지며, 시작(무력감)과 끝(해방감+여운)이 완결적인 미니 서사를 형성한다.

다만, "서사적 완결감"이라는 측면에서 이 장이 하나의 독립된 미니 서사 단위로서 자족적이라는 점에 대한 **명시적 언급**은 다소 부족하다. 장의 흐름상 암묵적으로 이해되지만, Episode 기능의 "자족적 구조"를 의식적으로 선언하면 더 강해진다. [+4]

**[B-5] 인과적 사건 연쇄 (4/4)**

사건의 인과 관계가 매우 촘촘하다:
- 3장 포획 -> 감금 -> 겨울이 추적 -> 구출 (지훈-겨울이 유대가 근거)
- 구출 후 입구 방향에서 발소리 -> 안쪽으로 피신 (수진 판단)
- 안쪽 이동 -> 인광 빛 발견 -> 호기심으로 심층부 진입 (호기심+생존 본능)
- 심층부 제단 발견 -> 민준 촬영 본능으로 빛 비춤 -> 거울 반사 -> 두루 소환 (촬영 집착이 촉매)
- 두루 교감 -> 규칙 발견 -> 수진 전략 제시 -> 악당 귀환 -> 전술적 탈출
- 마희재 끼임(체형이라는 물리적 조건) -> 거리 확보 -> 탈출 성공

우연 의존이 거의 없고, 각 사건이 캐릭터의 성격/능력에서 자연스럽게 흘러나온다. 겨울이의 구출도 1막부터 축적된 지훈-겨울이 유대의 서사적 결산으로 처리하여 인과적 설득력이 높다. [+4]

**[C-2] 역할 분배의 효율성 (4/4)**

7명의 캐릭터(수진, 지훈, 민준, 두루, 겨울이, 마기재, 마희재) 모두가 서사적으로 기여하는 바가 명확하다:
- 수진: 피신 판단, 전략 방향, 빛원 분담, 모티프 연결
- 지훈: 겨울이 교감, 두루 감정 읽기, 보조 라이트, 길 선택
- 민준: 촬영 장비로 두루 촉발, 초콜릿 교감, 거울 운반, 코미디
- 두루: 앵커 장면 중심, 세계관 규칙 체현, 추격전 지연
- 겨울이: 구출, 길 안내, 악당 유인
- 마기재: 추격 지시, 격분 약점, 거울 집착으로 3-4막 동기 예비
- 마희재: 선두 추격, 겨울이 멈칫, 좁은 통로 끼임

배경 인물로 전락하는 캐릭터가 없으며, 악당의 서사적 기능(위협 + 코미디 + 다음 막 동기)도 설계되어 있다. [+4]

**[C-3] 대사 패턴과 행동 일관성 (5/5)**

- **수진 (공적/사적 전환)**: "입구는 막혀 있어. 안쪽으로 가자"(공적 판단), "석등의 꽃받침 문양이랑 같아!"(분석적 관찰), "이 도깨비를 우리 편으로 만들 수 있어"(전략적 리더십). 3장의 떨리는 손(사적 노출) 이후 4장에서 판단력을 발휘하되 여전히 두려움이 공존하는 전환이 구체적으로 설계됨
- **지훈 (문자 그대로/자연 비유)**: "겨울이!" (직관적 감지), "별 같다" (자연 비유), "이 친구도 우리만큼 무서운 거야" (사회적 필터 없는 감정 읽기). 트리거와 발현 상황이 구체적
- **민준 (콘텐츠/촬영 중심)**: "카메라는 괜찮아?!" (촬영 장비 우선), "여기 찍어야 해!" (촬영 본능), "이 콘텐츠 대박이야!" (추격전 속 흥분), "배고프면 무서운 게 더 무서워. 유튜브에서 배웠어" (콘텐츠 출처 지식)

- **마기재/마희재**: 형제 투닥 패턴("이것도 제대로 못 하냐!" / "이게 들어가면 나도 좋겠다고!")이 악당의 입체성과 코미디를 동시에 제공

- 지훈: 사회적 필터 없는 직설(아이러니) -> 비인간 존재와의 교감(능력 확장). 겨울이 -> 두루로 교감 대상이 격상되는 패턴

**[C-5] 캐릭터 상호작용 설계 (4/4)**
