## 정합성 체크리스트 보강 항목

2. 마을 인프라: 학원/병원/편의점 등이 마을 규모와 부합하는지
3. 소품 시대 적합성: 2020년대 기술 소품 확인 (삼각대->짐벌 등)
4. 감각 묘사 물리적 타당성: 냄새/소리의 도달 거리, 시간대 적절성
5. 신체 묘사 연령 적합성: 13-14세 아이의 체격 묘사 과장 금지

3. 주어 생략 문장의 귀속 명확성

## 비트 설계 원칙 보강

### 감정 반복 방지
- 유사 감정 비트 연속 시 각 비트의 차별점을 한 줄 요약 필수
- "시작", "출발", "결의" 등 동의어 반복은 씬 전체에서 1회 제한
- 예: "Beat 1: 결의의 **선언** / Beat 2: 안도의 **이완** / Beat 3: 불안의 **떨림**"

### 전환 트리거 강화
- 모든 전환에 캐릭터 행동 또는 외부 사건이 트리거로 필수
- "갑자기", "그때", "바로 그순간"만으로는 불충분

### 감각 묘사 기능 명시
- 형식: `[감각: 시각-OOO / 기능: 시간경과 표시]`
- 기능 4종: (1) 분위기 설정, (2) 시간 경과, (3) 복선 설치, (4) 감정 반영
- 기능 없는 감각 묘사 = 소설에서 "의미 불명의 묘사"로 전락

## 대사 작성 가이드 보강

### 민준 대사 변주
- 단순 감탄(대박/채택)보다 상황을 콘텐츠 용어로 재해석하는 구체적 대사
- 나쁜: "자, 카메라는 계속 돌아갑니다!" -> 좋은: "마을 진입! 로케이션 헌팅 가봅시다!"

### 수진 카메라 회피
- 외모 변명("머리 안 감았는데")보다 진짜 동기(비밀 보호) 연결, 또는 대사 없이 행동만

### 다의성 처리
- 핵심 수수께끼가 다의적일 때 캐릭터가 다의성을 먼저 인지하게 하여 독자 혼란 방지
- "꽃이 피는 돌" = "돌이 꽃을 피운다" vs "돌에서 꽃이 피어난다" 양쪽 모두 제시

## 내부 가이드 vs 소설 표현 구분
- 내부 가이드: `**캐릭터 특성 발현**: (설명)` -- 작가 참고용
- 소설 방향: `**표현 방향**: (행동/대사 예시)` -- 소설 반영용
- 설정 라벨("완벽한 학생회장")은 절대 소설에 직접 사용 금지
