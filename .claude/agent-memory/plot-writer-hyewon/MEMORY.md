## 복선 vs 모티프 구분
- 모티프: 반복되는 상징적 요소 (청동 거울, 꽃이 피는 돌 등)
- 복선: 향후 사건을 미리 암시하는 장치 (민준 초콜릿 = 욕심 없는 자 등)

## 팀원 관점
- 박호진(집필): 공간 동선, 시간축, 감각 정보, 캐릭터 상호작용
- 이연수(편집): 독자 경험, 횡단 평가, 설정 충돌 해결
- 김태수(팀장): 문서 비대화 경계, 기존 체계 통합 선호

## 핵심 패턴 요약 (상세: evaluation-patterns.md)
- **초안 공통 결함**: 공간 동선/시간 프레임/성장 아크/유머/연령/독자경험 누락 -> 체크리스트화 필요
- **정보 손실**: Act->Chapter->Scene 각 단계에서 반복. 상위 A등급이어도 하위 자동 우수 아님
- **D항목 약진**: 1차 최하 -> 2차 최대 상승 패턴 반복
- **3중 태그**: 감정+감각+기능 = Scene 문서 표준
- **전환 설계**: "서사적 계기 + 감각적 변화" 복합 구조
- **종료 후크**: 막/장 마지막에서 최우선 검증. 누락 시 5항목 동시 영향
- **"갑자기"**: 인과 트리거 부재의 신호
- **피드백 전수 수용**: D등급 초안에서는 거절보다 전수 수용이 효율적
- **에필로그 고유**: 전환점은 사건이어야, 04-act 미해결 과제 매핑 필수, 마지막 이미지 팀 결정
