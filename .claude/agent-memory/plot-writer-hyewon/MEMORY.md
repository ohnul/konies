# 조혜원 플롯작가 에이전트 메모리

## 프로젝트 핵심 구조
- Act-Chapter-Scene 3단 구조. Beat/Moment는 Scene에 흡수, Sequence/Episode는 Chapter에 흡수.
- 6개 Act: 프롤로그, 1-4막(기승전결), 에필로그
- 분량 비중: 프롤로그 5%, 1막 20%, 2막 30%, 3막 25%, 4막 15%, 에필로그 5%

## 평가 기준 문서 히스토리
- `act/act-evaluation-criteria.md`: 2차 리뷰 완료 (2026-02-07). A등급 판정.
- 내 1차 피드백에서 수용된 것: 세부 배점, 복선 추적 표, D-6, E-6, A-2 보강, 공간 동선/시간 프레임
- 미수용: F항목 독립, 정보 공개 전략(Chapter에서 다룰 예정), 캐릭터 등장 4단계, 평가 예시 확장
- 잔여 과제: D-6/E-6 배점 적정성은 실제 평가 후 재검토, 정보 공개 전략은 Chapter 평가 기준에서 반영 필요

## 피드백 작성 패턴
- 2차 리뷰는 1차보다 짧고 핵심만. 수용 확인 + 반영 품질 + 잔여 이슈 구조가 효과적.
- 피드백 파일 네이밍: `{날짜}-{시간}-{대상}-feedback-{작성자}.md`

## 팀원 관점 메모
- 박호진(집필): 공간 동선, 시간축, 감각 정보, 캐릭터 상호작용에 집중. "살붙이기에 적합한 뼈대" 관점.
- 이연수(편집): 독자 경험, 횡단 평가 위상, 설정 충돌 해결, 프롤로그/에필로그 형식 예외에 집중.
- 김태수(팀장): 문서 비대화 경계, 기존 체계에 통합 선호, Chapter 평가 기준으로 후속 대응 패턴.

## 복선 vs 모티프 구분 (프로젝트 공식 정의)
- 모티프: 반복되는 상징적 요소 (청동 거울, 꽃이 피는 돌, 구름골 등)
- 복선: 향후 발생할 특정 사건을 미리 암시하는 장치 (민준 초콜릿 = 욕심 없는 자 등)
