## 1막 NOTE-TO-AI 메모 (작가 내부 의견)

## 복선 vs 모티프 구분 (프로젝트 공식 정의)
- 모티프: 반복되는 상징적 요소 (청동 거울, 꽃이 피는 돌, 구름골 등)
- 복선: 향후 발생할 특정 사건을 미리 암시하는 장치 (민준 초콜릿 = 욕심 없는 자 등)

## 피드백 작성 패턴
- 2차 리뷰는 1차보다 짧고 핵심만. 수용 확인 + 반영 품질 + 잔여 이슈 구조가 효과적.
- 재평가서는 "이전 평가 대비 변경 사항 요약"으로 시작하면 명확.
- 피드백 파일 네이밍: `{날짜}-{시간}-eval-{대상}-{작성자}.md`

## 팀원 관점 메모
- 박호진(집필): 공간 동선, 시간축, 감각 정보, 캐릭터 상호작용에 집중.
- 이연수(편집): 독자 경험, 횡단 평가 위상, 설정 충돌 해결에 집중.
- 김태수(팀장): 문서 비대화 경계, 기존 체계 통합 선호, Chapter 평가 기준으로 후속 대응.

## Chapter 평가 시 발견한 패턴
- Act 문서가 우수해도 Chapter 문서가 자동으로 우수하지 않음. Act -> Chapter 구체화 단계에서 정보 손실 발생 가능
- 단일 씬 구성은 시공간 논리로는 정당화되지만 서사 부하 관점에서 위험. Chapter에서 씬 내부 Beat/Moment 가이드 필수
- D항목(톤/페이싱)은 Act에는 풍부하나 Chapter에서 누락되기 쉬운 영역. Chapter 문서 작성 시 Act의 D 관련 내용을 적극 하향 전개해야 함
- **모멘트 구조화 패턴**: 단일 씬에서 5개 모멘트에 각각 비중%/긴장도/캐릭터/전환트리거/감각키워드/유머배치를 부여하는 방식이 매우 효과적. 01-01-chapter 개정판이 모범 사례. 이후 단일 씬 Chapter에서 참조할 것
- **인지 부하 관리 3요소**: (1) 캐릭터 단계적 투입, (2) 태그 행동으로 기억 보조, (3) 핵심 정보 다관점 반복. 5명 동시 소개가 필요한 설정 장에서 특히 유효
- **재평가 시 주의**: 이전 평가 점수와 비교할 때, 동일 점수여도 질적 향상이 있을 수 있음(D-6 사례). 점수 변화가 없는 항목도 내용적 변화를 확인해야 함
- **평가 예시와 독립 평가의 차이**: chapter-evaluation-criteria.md의 평가 예시(C등급 67점)와 내 독립 평가(D등급 56점) 사이에 11점 차이 발생. 원인: (1) A-4 참조 문서명 갱신 미반영은 예시에서 미발견, (2) E-3 시간 프레임 불일치는 예시에서 경미하게 처리됨, (3) A-3 미니 클라이맥스 미명시에 대해 예시(3점)보다 엄격(2점). 평가 예시는 참고자료이며 독립 평가를 대체하지 않는다는 점 재확인
- **"분량이 큰 장 = 가이드도 풍부해야" 원칙**: 1막 2장 개정판에서 이 원칙이 성공적으로 적용됨. 70% 비중 장이 30% 비중 장보다 3점 높은 결과
- **D항목 약진 패턴**: 1차에서 가장 낮은 항목(D)이 2차에서 가장 높은 항목이 되는 패턴이 1장과 2장 모두에서 반복됨. Chapter 문서 작성 시 D항목(톤/유머/감각/리듬)을 처음부터 설계하면 개선 비용이 줄어듦
- **4개 씬 구성의 이점**: 4개 씬 구성이 단일 씬보다 D항목(씬별 톤/감각/유머 배치)과 A항목(비중 표시, 방향 제시)에서 유리. 01-02-chapter(4씬, 88점) vs 01-01-chapter(1씬, 85점)
- **미니 클라이맥스 씬 경계 문제**: 미니 클라이맥스가 두 씬에 걸쳐 있으면 Scene 문서 작성 시 혼란 가능. 정점을 하나의 씬에 명확히 귀속시키는 것이 바람직

## Scene 평가 기준 피드백에서 발견한 패턴
- **A-3(구조)/B-2(내용) 이중 감점 위험**: 비트/모멘트의 "존재와 분류" vs "감정 추적의 질적 효과"의 경계가 실제 평가에서 흐려짐. 평가 예시에서 이미 이중 감점 발생. "라벨 존재 여부" vs "라벨 내용의 효과"로 구분 권고
- **"therefore/but" 원칙의 비트 단위 한계**: 비트 단위에서는 "정당한 and then"(탐색 시행착오의 나열 등)이 존재. 누적적 효과가 설계 의도로 명시되면 허용해야
- **D-6 페이싱 가이드 변별력 부족**: 2점 배점에 스케일 앵커 부재. 모멘트 수준 가이드(있음)와 비트 수준 가이드(없음)의 차이가 0점 vs 1점의 차이인지 불명확
- **Scene 수준에서 "독자 지식 상태 추적"이 가장 실효적**: Chapter에서 보류 -> Scene에서 검토 약속 -> 미반영. 정보 공개의 실행 단위인 Scene에서 이 추적이 가장 효과적
- **01-02-02-scene.md의 "연꽃 문양" 설정 충돌**: 108행 지훈 대사에 여전히 "연꽃 문양" 잔재. Scene 평가 예시에서 정확히 식별되었으나, 실제 Scene 문서 수정은 미완
