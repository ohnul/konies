## 4-Evaluator Feedback Review Process
- Workflow: Read all 4 -> consolidate unique items -> classify accept/hold/reject -> write report -> apply accepted -> respond to each evaluator
- Common items (3+/4 agreement) auto-accepted. Disagreements resolved by editorial judgment.
- Score ranking pattern: 연수 > 혜원 > 호진 >= 태수 (consistent across evaluations)
- Higher-level docs show more evaluator divergence (Acts 6-12pts) than Chapters (3-7pts)
- Zero rejections possible when all evaluators converge (03-act case)
